# SnarkyJS

## SnarkyJS - Web

```
npm install
npm run start:web
```

## SnarkyJS - Server

```
npm install
npm run start:server
```

## Run Tests

```
npm run test
```

## Publish

```bash
# Make sure to have proper NPM credentials before publishing.
# To get credentials, use `npm login`
./scripts/publish.sh
```
